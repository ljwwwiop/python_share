功能说明：
	将各功能定义成函数模块
	1. file_format： 文件格式化成字典
	2. file_write:   将字典重新写入文件
	3. select:		 查询员工工资模块
	4. change:		 修改员工工资模块
	5. add:			 添加新员工记录模块


	
实现效果：
从info.txt文件中读取员工及其工资信息，最后将修改或增加的员工工资信息也写入原info.txt文件。
